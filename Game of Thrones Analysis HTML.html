Analyzing the Book: Game of Thrones¶

This text file 'gameofthrones' contains books 1-5 from the Game of Thrones Boxed Set, including 'A Game of Thrones,' 'A Clash of Kings,' 'A Storm of Swords,' and 'A Feast for Crows.'" Our objective in this analysis is to identify the top 10 main characters in the book series.¶

First step, we read in the file¶

In [2]:
with open('gameofthrones.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()
    print("File read in successfully.")

File read in successfully.

Next step is counting the frequency of each word, line by line.¶

In [3]:
d = {}

for l in lines:
    l = l.strip()
    if not l:
        continue 
    words = l.split()
    caps = [w.rstrip('.’,;?!:"') for w in words if w[0].isupper()]
    caps = [w.replace("\'s", '') for w in caps]
    for n in caps:
        if n in d: 
            d[n] += 1
        else:
            d[n] = 1

mains = sorted(d.items(), key=lambda x: x[1], reverse=True)
